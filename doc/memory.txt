0x00000-0x3FFFF -> Read Only Cartridge

0x40000-0x403FF -> Hardware config registers
    Control -> 0x4
    Music   -> 0x8
    Palette -> 0x30

0x40400-0x439FF -> Screen buffer

0x43A00-0x46FFF -> 4 sprite sheet

0x47000-0x4DBFF -> 4 tilemaps

0x4DC00-0x4DFFF -> Save Area

0x4E000-0x4F7FF -> Empty

0x4F800-0x4FFFF -> Multicart
